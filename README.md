# Healthcare Web Application

This project is a scalable, secure healthcare web application for fibromyalgia prediction and teleconsultation, featuring patient and doctor dashboards, secure authentication, encrypted data, payment integration, notifications, chatbot, video consultation, audit logging, and accessibility.

## Main Features
- Patient and doctor registration/login
- JWT/OAuth2 authentication
- Doctor verification (admin approval, document upload)
- Fibromyalgia prediction and report generation
- Appointment scheduling and payment (RazorPay)
- Patient-only chatbot (context-aware, FAQ, handover to human)
- Video consultation (Twilio/Zoom)
- Doctor dashboard (profile, availability, notes)
- Patient dashboard (history, reports, feedback)
- Notifications (email/SMS)
- Audit logging
- Accessibility and mobile-friendly UI

## Tech Stack
- Backend: FastAPI (Python)
- Frontend: React
- Database: PostgreSQL
- Caching/Session: Redis
- Payment: RazorPay
- Notifications: Email/SMS API
- Video: Twilio/Zoom

## Setup
Each library and dependency will be installed in the respective backend/frontend directories using virtual environments and local node_modules.
